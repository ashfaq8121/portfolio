# DESIGN.md

## Design Approach
I kept the design simple and clean. Dark background with light text. The goal was to look professional without being overcomplicated.

## Colors
- Background: dark navy `#0f0f0f`
- Text: white `#f0f0f0`
- Accent: blue for links and buttons
- Cards: slightly lighter surface color

## Typography
- System fonts only — no Google Fonts, faster loading
- Large headings, comfortable line height for reading

## Layout
- Max width 72rem, centered
- Fixed navigation bar at top
- Cards for projects and blog posts
- Two column layout on desktop, single column on mobile

## Dark Mode
- Default is dark mode
- Toggle button in navigation
- Preference saved in localStorage

## Mobile
- Hamburger menu on mobile
- All pages tested at 320px minimum width
- Touch friendly buttons and links

## Logo
- AUR monogram — my initials (Ashfaq Ur Rahman)
- Crescent moon design representing my name
